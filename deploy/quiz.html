객관식
통계학의 세 가지 핵심 역할을 올바르게 나열한 것은?
  • 데이터 수집 → 가설 검증 → 결과 발표
  • 데이터 수집 → 데이터 요약 → 통계적 추론
  • 표본 추출 → 그래프 작성 → 모수 결정
  • 실험 설계 → 데이터 정제 → 머신러닝
해설
통계학은 데이터 수집(Collection) → 데이터 요약(Summarization) → 통계적 추론(Inference)의 과정으로 정의됩니다.

수집 단계에서는 현상을 왜곡 없이 반영하는 데이터를 모으고, 요약 단계에서는 기술통계를 사용해 패턴을 파악하며, 추론 단계에서는 표본으로부터 모집단에 대한 결론을 도출합니다.
2
객관식
어떤 조사에서 전국 대학생 10,000명의 월평균 생활비를 알기 위해 500명을 무작위로 선정해 조사했다. 이때 모수(parameter)에 해당하는 것은?
  • 조사에 응답한 500명의 평균 생활비
  • 500명을 뽑는 방법
  • 전국 대학생 10,000명의 실제 평균 생활비
  • 조사 결과로 만든 그래프
해설
모수(parameter)는 모집단 전체의 특성값으로, 고정된 값이지만 대부분 알 수 없습니다.

여기서 모집단은 전국 대학생 10,000명이고, 그들의 실제 평균 생활비가 모수입니다. 반면 500명에서 계산한 평균은 통계량(statistic)으로, 어떤 표본을 뽑느냐에 따라 달라집니다.
3
주관식
데이터의 세 가지 기본 요소를 각각 한 단어로 쓰시오. (쉼표로 구분)

해설
정답: 단위, 변수, 관찰값

데이터를 표(table)로 생각하면 이해가 쉽습니다.
- 단위(Unit): 관찰 대상 하나하나 (표의 각 행)
- 변수(Variable): 측정하는 특성 (표의 각 열)
- 관찰값(Observation): 특정 단위에서 특정 변수를 측정한 값 (표의 각 셀)
4
주관식
단순랜덤표집(Simple Random Sampling)이 왜 중요한지, 그리고 이를 사용하지 않을 때 발생할 수 있는 문제를 한 문장으로 설명하시오.

모범답안
단순랜덤표집은 모집단의 모든 부분집합이 동일한 확률로 선택될 수 있도록 보장하여, 표본이 모집단을 편향 없이 대표하게 합니다. 이를 사용하지 않으면 특정 집단이 과대/과소 대표되어 편향된 결론을 내릴 수 있습니다.

예를 들어 서울 강남 한 지역에서만 주거비를 조사하면, 전국 평균보다 높은 결과가 나올 수밖에 없습니다. 단순랜덤표집은 이런 선택 편향(selection bias)을 방지합니다.
5
객관식
다음 중 기술통계(descriptive statistics)에 해당하는 것을 모두 고르면?

ㄱ. 표본의 평균과 표준편차를 계산한다
ㄴ. 표본 결과로 모집단의 평균을 추정한다
ㄷ. 데이터를 히스토그램으로 시각화한다
ㄹ. 신뢰구간을 구한다
  • ㄱ, ㄴ
  • ㄱ, ㄷ
  • ㄴ, ㄹ
  • ㄱ, ㄴ, ㄷ
해설
정답: ㄱ, ㄷ

기술통계는 주어진 데이터를 정리하고 요약하는 것입니다. 평균/표준편차 계산(ㄱ)과 그래프 작성(ㄷ)이 여기에 해당합니다.

반면, 표본으로 모집단을 추정하는 것(ㄴ)과 신뢰구간(ㄹ)은 추론통계(inferential statistics)에 해당합니다. 추론통계는 표본에서 모집단으로 일반화하는 과정입니다.
