description: 사용자가 질문한 내용과 답변을 study_guide.html에 Q&A 섹션으로 추가. 의문을 가졌던 것을 반복 학습하면 기억에 잘 남기 때문.
type: feedback

사용자가 개념에 대해 질문하면, 답변 후 해당 Q&A를 study_guide.html의 관련 강 섹션에 추가한다.

**Why:** 사용자가 직접 의문을 가졌던 내용을 정리해서 반복 열람하면 기억에 더 잘 남는다고 함.

**How to apply:** 질문이 들어오면 (1) 대화에서 먼저 답변 (2) study_guide.html의 해당 강 위치에 Q&A 박스 형태로 삽입. 질문 원문과 핵심 답변을 포함.
