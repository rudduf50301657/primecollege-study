**4강 확률:**
- [10, 30] 구간 표기 vs {1,2,3} 나열 표기의 차이 (이산형 vs 연속형 표본공간)
- 연속형 확률의 "길이 비율" 계산법 (균등분포일 때만 성립)
- "측도" 용어 = 이산형은 개수, 연속형은 길이
- 확률의 3가지 정의 (고전적/상대도수적/공리적), 특히 공리3 배반사건
- 순열에서 (n-r)!로 나누는 이유 (뒷부분 잘라내기)
- 0! = 1인 이유
- 조합에서 r!로 추가로 나누는 이유 (순서 중복 제거)
- P(A∩B) ≠ P(A)×P(B) — 교집합은 곱하기가 아님!
- P(A∩B) vs P(A|B) — 분모(기준)가 다름
- "|" 기호 = "~일 때", 조건부확률에서 세상을 축소
- 조건부확률 예제에서 직관(8/20)과 공식의 관계
- "사상" = "사건"의 다른 표현, "곱사건" = A∩B

**5강 확률변수:**
- 확률변수 X는 문제에서 정의해줘야 함 (자동으로 정해지지 않음)
- 확률분포함수 vs 누적확률분포함수 (산 모양 vs S자)
- 이산형 확률분포의 3가지 성질, 부등호 < vs ≤ 차이
- E(X) 기호들 읽는 법 (μ, Σ, xᵢ, f(xᵢ))
- E(X²)는 분산 간편공식의 재료 ("제기 − 기제")
- E[...]는 "기댓값 구해라"라는 명령어 (자동으로 ×f(x)하고 Σ 포함)
- [E(X)]² ≠ E(X²) 주의!
- 기댓값 = 이론적 평균 (미래에 대한 평균, 확률을 곱하는 이유)
- aX+b에서 a=스케일(확대/축소), b=위치(이동)
- 표준편차 = 떨어진 정도의 (대략적) 평균
- 편차 합이 항상 0인 이유 → 제곱을 쓰는 이유
- 치우친 분포에서도 편차 합 = 0 (평균이 끌려가서 균형 맞춤)
