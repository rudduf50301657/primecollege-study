- 통계학개론 수강 중 (프라임칼리지, 박서영/이기재 교수)
- 수학 기호(Σ, μ, σ 등)에 익숙하지 않음 → 기호를 만나면 읽는 법 + 한국어 번역 필요
- 공식만 나열하면 이해 어려움 → 반드시 예시와 비유로 설명해야 함
- "왜?"를 자주 물어봄 → 공식의 유도 과정/이유를 중시
- 강의를 들으면서 동시에 질문하는 스타일 → 강의 스크린샷을 보내주며 질문
- 직관이 좋음 (조건부확률 8/20을 바로 파악, 문제의 한계를 지적하기도 함)
